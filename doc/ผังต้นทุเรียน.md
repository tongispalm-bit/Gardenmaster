# หน้าสวน — Farm Map ผังสวนทุเรียนแบบ Interactive

---

## 1. Grid Layout 9×11

- แสดงผังต้นทุเรียนแบบ Grid 9 คอลัมน์ × 11 แถว
- Label คอลัมน์ C1–C9 (แนวนอน) และแถว R1–R11 (แนวตั้ง)
- รหัสต้นเริ่มต้นแบบ `B0304` = แถวที่ 3 คอลัมน์ที่ 4 (zero-padded)

---

## 2. กฎการแสดงผลช่อง (Cell Rules)

- **R1C1–R1C8** → ว่าง (ไม่มีต้น)
- **R1C9** → มีต้น
- **R2–R9, C1–C9** → มีต้นครบทุกช่อง
- **R10–R11, C1–C9** → มีต้นครบทุกช่อง
- รวมต้นทั้งหมด **91 ต้น**

---

## 3. Color-Coded Status

| สี | ความหมาย | ไอคอน |
|---|---|---|
| พื้นเขียวอ่อน | ปกติ | 🌳 |
| พื้นแดงอ่อน | เฝ้าระวัง / มีโรค | 🌲 |
| พื้นฟ้าอ่อน | ต้นกล้า / ปลูกใหม่ | 🌴 |

---

## 4. Summary Header

- การ์ดสรุป 4 ช่อง: ต้นทั้งหมด / ปกติ / เฝ้าระวัง / ต้นกล้า
- ตัวเลขอัปเดต real-time ทุกครั้งที่บันทึกข้อมูล

---

## 5. Interactive Hover

- เอาเมาส์ชี้ที่ต้น → ขยาย `scale(1.35)`
- Tooltip แสดง: รหัสต้น · พันธุ์ · สถานะ

---

## 6. Modal แก้ไขรายต้น

กดที่ต้นไม้ในผัง → เปิด modal พร้อมฟิลด์แก้ไขได้ 5 รายการ:

| ฟิลด์ | รายละเอียด |
|---|---|
| รหัสต้น | แก้ไขได้อิสระ สูงสุด 20 ตัวอักษร เช่น T-001, A0101 |
| สถานะสุขภาพ | Dropdown: ปกติ / เฝ้าระวัง / ต้นกล้า |
| พันธุ์ | Dropdown 8 พันธุ์: หมอนทอง / ชะนี / กระดุม / พวงมณี / ก้านยาว / มูซานคิง / โอฉี / นวลทองจันทร์ |
| อายุต้นทุเรียน | กรอกตัวเลขอายุ (ปี) |
| หมายเหตุ | Textarea พิมพ์อิสระ |

---

## 7. Validation รหัสต้น

- ป้องกันรหัสซ้ำ — แจ้งเตือนทันทีหากรหัสซ้ำกับต้นอื่น ไม่บันทึกจนกว่าจะแก้ไข
- ป้องกันรหัสว่าง — บังคับกรอกรหัสทุกครั้ง

---

## 8. Real-time Update

- กด "บันทึก" → รหัส สี ไอคอน และ Summary Header อัปเดตบนผังทันที
- กด "ยกเลิก" หรือคลิกนอก Modal → ปิดโดยไม่บันทึก

---

## 9. Tech Stack

- HTML / CSS / Vanilla JS (ไม่มี dependency ภายนอก)
- Font: Sarabun (Google Fonts) สำหรับภาษาไทย
- ใช้ CSS Variables สำหรับ theming (รองรับ light/dark mode อัตโนมัติ)
