ข้อมูลที่แนะนำให้มีใน Header

  • ชื่อสวน + ไอคอน — ระบุว่ากำลังอยู่สวนไหน (มีทุกแบบ)
  • ปุ่มย้อนกลับ / หน้าแรก — นำทาง (มีทุกแบบ)
  • สลับธีมสว่าง/มืด — ใช้บ่อย ควรเข้าถึงได้เสมอ
  • รอบปี พ.ศ.เก็บเกี่ยว — สำคัญมากเพราะข้อมูลผูกกับรอบปี (แบบ 7, 10)
  • สรุปจำนวนต้น/สถานะ — รวม/ปกติ/เฝ้าระวัง/รักษา ดูภาพรวมไว (แบบ 6, 10)
  • แจ้งเตือนต้นป่วย — เด่นชัดเมื่อมีต้นกำลังรักษา (แบบ 8)
  • สถานะ sync/ออนไลน์ — ให้มั่นใจว่าข้อมูลถูกบันทึกแล้ว (แบบ 7)
  • สุขภาพสวน % — ตัวชี้วัดรวมแบบเห็นง่าย (แบบ 9)
  • Breadcrumb — บอกตำแหน่งในเมนูย่อย (แบบ 9)
